# 程序的实现的功能

1. find
   - 实现了查找功能，包括精确查找和模糊匹配
   - 可以查询所有字段和任意指定的字段
2. add
   - 实现了添加用户的功能
   - 可以在添加用户时验证用户手机号是否存在
3. del
   - 实现了删除用户的功能
   - 可以一次删除多个用户
4. update
   - 实现了编辑用户的功能
   - 可以一次编辑多个用户多个字段
